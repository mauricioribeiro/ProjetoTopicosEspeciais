# ProjetoTopicosEspeciais

----
## Back-End endpoints

### POST /login
Autentica o usuário e retorna um token de autentificação que precisará ser usado no header:
> Authorization: Bearer %token%

### GET /usuarios
Lista os usuários
 * Nome
 * Email
 * Permissões

### GET /permissoes
Lista permissões
 * Nome
 * Página
